## 📋 What is chrome-pdf-to-notebooklm?

chrome-pdf-to-notebooklm is a Chrome extension that lets you turn any PDF document into a NotebookLM notebook. It handles the entire process automatically. After conversion, it also creates extra files like an audio overview. This tool works inside your browser, so there's no need for complex setups or programming knowledge.

The extension is useful for anyone who reads PDFs and wants a quick way to convert notes and documents into a format that NotebookLM can manage. It saves time and makes reviewing important points easier.

## 🔧 Troubleshooting

- Extension icon not showing: Make sure Chrome is updated and Developer Mode is enabled during install.
- PDF won’t upload: Check the file size. Large files over 100MB may fail.
- Conversion takes too long: Try closing unnecessary tabs or programs. The process needs computer resources.
- Audio not playing: Check your sound settings and ensure you have speakers or headphones connected.
- NotebookLM not opening: Confirm you have access to NotebookLM or the correct app installed.

If issues persist, uninstall the extension and reinstall it following the steps above.

## 🧩 How It Works Internally

chrome-pdf-to-notebooklm uses Chrome APIs to interact with PDF files directly. It reads your PDF content and extracts text, images, and layout information. Then, it formats this data into a NotebookLM-compatible structure.

The extension creates an audio summary using text-to-speech features, allowing you to listen to the key points without reading everything.
